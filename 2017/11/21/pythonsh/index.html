Python SH模块

Nov 21, 2017 #Python

一个在Python脚本中方便处理bash命令的模块

一、介绍

  • 这是一个在Python脚本中方便处理bash命令的模块，使用方便，在代码中可以让你的shell交互变得更加简单。
  • sh将系统的命令动态映射到python函数，通过python的方式去写shell脚本

二、安装

  1. 使用pip执行sh模块安装
pip install sh

三、基本使用

  1. 参数传递

    # 查看/home目录
import sh
sh.ls("/home","-l")
# 输出结果
总用量 12
drwx------ 2 luochen luochen 4096 10月 17 14:16 luochen
drwx------ 3 Magkiim Magkiim 4096 10月 17 23:41 Magkiim
drwx------ 2 ts3srv  ts3srv  4096 10月 17 13:37 ts3srv

  2. 关键字参数

    • 当命令里需要参数时，sh调用表现方式会与期望一样
    # 常用shell命令
curl https://www.baidu.com -o page.html --silent
# 使用python sh模块
import sh
sh.curl("https://www.baidu.com",o="page.html",silent=True)

  3. 查找命令

    • 检查命令是否存在，若不存在则执行安装操作
    import sh
if not sh.which('supervisorctl'):
    sh.apt_get("install","supervisor","-y")

  4. 调用脚本

    import sh
r = sh.Command("/data/test.py")
# 获取输出结果
r()

  5. Baking命令

    • 类似于函数绑定，将参数绑定到函数上
    from sh import ls
ls = ls.bake("-la")
# 实现默默加上选项-la
print ls("/home")

  6. glob列出文件,获取列表

    import sh
filelist = sh.glob("/data/*")
print filelist
['/data/python', '/data/database', '/data/swap', '/data/logs', '/data/save', '/data/golang', '/data/shell', '/data/conf', '/data/docker', '/data/tmp', '/data/backup']

  7. 命令管道

    # 常用shell命令
ls /data | wc -l
# 使用python sh模块
import sh
sh.wc(sh.ls("/data"),"-l")
# 管道是有序执行的，默认是从内到外的，若需要并行，可加_piped=True

  8. 重定向输出

    # 将命令结果重定向输出
import sh
sh.ls("/data",_out="/data/tmp/test_result.txt")
# 等价于
with open("/data/tmp/test_result.txt","w") as h:
    sh.ls("/data",_out=h)

  9. 实时迭代输出

    from sh import tail
# 执行tail -f追踪日志
for line in tail("-f","/var/log/messages",__iter=True):
    print(line)

四、异常捕捉

  • 当进程退出码非0时，将会抛出异常。可以捕捉特殊返回码或者通过基础异常类捕捉
import sh
try:
    print(sh.ls("/data"))
except ErrorReturnCode_2:
    print("folder doesn't exist.")
except ErrorReturnCode:
    print("unknown error")
