Centos6 挂载16T硬盘

Jul 4, 2017 #Linux

Parted工具分区与更改分区格式GPT

前提

  • 由于MBR分区支持最大容量只有2T，如果超过2T的话，就需要使用gpt分区

将硬盘转换成gpt分区

parted /dev/sdb
mklabel gpt 
# 由MBR格式转换成GPT格式需要删除所有数据，慎重使用！

使用parted划分分区

# 划分所有分区
mkpart primary 0 -1
# 0,-1表示起点容量和终点容量
# 删除分区
rm ${ID}
# 退出
quit

使用ext4进行格式化

mkfs.ext4 /dev/sda4

检查挂载情况

df -Th
# -T 会输出磁盘的格式
