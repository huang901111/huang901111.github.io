Python Hashlib模块

Jul 4, 2017 #python

Hashlib模块提供了常见的摘要算法，如MD5，SHA1等等

一、简介

  • Python中的hashlib模块是用来进行hash或者md5加密。
  • 这里的加密，其实并非我们通常所说的加密，简单的说就是这种加密一般是不可逆的。这种加密算法实际上是被称为『摘要算法』。
  • 摘要函数，又称为哈希算法、散列算法，它通过一个函数，把任意长度的数据转换为一个长度固定的数据串（通常用16进制的字符串表示）。
  • 摘要函数可以用来检验数据是否发生改变。

总结：hashlib是个专门提供hash算法的库，里面包含md5、sha1、sha224、sha256、sha384、sha512。

二、快速入门

2.1 hashlib对象常用方法

  • update(arg)：使用string arg更新hash对象，多次调用update和使用串联的参数一次调用update是等价的。
    • m.update(a);m.update(b) 等效于 m.update(a+b)
  • digest()：返回二进制字符串。
  • hexdigest()：返回十六进制字符串。
  • copy()：返回hash对象的一个克隆，这可用于高效的计算共享一个相同初始字符串的digests字符串。
  • hashlib.algorithms：返回保证这个模块可以支持的算法tuple
  • hashlib.algorithms_guaranteed：返回保证这个模块可以在所有平台都支持的算法的set
  • hashlib.algorithms_available：返回当前环境支持的算法名
  • hash.name：使用算法的名称

2.2 基本例子

import hashlib

word = "GuangZhou Beijing"
method = hashlib.md5()  # 创建hash对象
method.update(word) # 更新哈希对象以字符串参数
print method.digest() # 返回摘要，作为二进制数据字符串值
print method.hexdigest()  # 返回十六进制数字字符串
print method.digest_size  # 返回产生的散列的字节大小
print method.block_size # 返回产生的散列的块大小

三、计算字符串摘要的三种方法

方法一

import hashlib

words = "I am a boy"
m = hashlib.md5()
m.update(words)
print m.hexdigest()
9901dc4f248dc9ee15f9951221d766a5

方法二

import hashlib

words = "I am a boy"
print hashlib.new("md5",words).hexdigest()
9901dc4f248dc9ee15f9951221d766a5

方法三

import hashlib

words = "I am a boy"
print hashlib.md5(words).hexdigest()
9901dc4f248dc9ee15f9951221d766a5

四、计算文件的摘要

小文件的简单MD5校验

import hashlib

def md5sum(filename):

  fp = open(filename,'rb')
  content = fp.read()
  fp.close()
  m = hashlib.md5(content)
  file_md5 = m.hexdigest()
  return file_md5
  
if __name__ == "__main__":
  print md5sum("xxxx.jpg")

缺点：由于是要检验文件里面的内容，所以每次是把文件读到内存的，如果文件很大，可能程序直接卡死。
