Leanote开源笔记

Jul 4, 2017 #interest

蚂蚁笔记，有极客范的云笔记！

环境介绍

  • 树莓派安装 Centos 7 arm
  • 利用官网 二进制包 实现部署
  • 具体参考教程：https://github.com/leanote/leanote/wiki/leanote-binary-installation-on-Mac-and-Linux-(En)

一、源码包下载

  • 下载链接：http://leanote.org/#download（选择与系统版本适合的包）
cd /data/backup/tmp
wget https://sourceforge.net/projects/leanote-bin/files/2.4/leanote-linux-arm-v2.4.bin.tar.gz/download
tar -zxf leanote-linux-arm-v2.4.bin.tar.gz 
cp -a leanote /data/

二、Mongodb部署

  • 需要安装arm版的mongodb

安装Mongodb

cd /data/backup/tmp
git clone https://github.com/Barryrowe/mongo-arm.git
groupadd mongodb
useradd -s /sbin/nologin -g mongodb -m mongodb
cp -a mongo-arm/binaries/2.1.1  /data/mongodb
chown -R mongodb:mongodb /data/mongodb
# 创建mongodb数据库保存目录
mkdir /data/database/mongodb
chown -R mongodb:mongodb /data/database/mongodb
# 添加环境变量
vim /root/.bashrc
export PATH=$PATH:/data/mongodb/bin
source /root/.bashrc

Mongodb测试

mongod --dbpath /data/database/mongodb &
# 进入mongodb
mongo
> show dbs
show dbs
admin   (empty)
local   0.03125GB

三、导入Leanote数据

mongorestore -h localhost -d leanote --dir/data/leanote/mongodb_backup/leanote_install_data/
# 导入完毕后，检查
mongo

> show dbs

五、Leanote测试

六、supervisor后台

pip install supervisor
# 生成配置
echo_supervisord_conf>/etc/supervisord.conf
# 添加配置
[program:mongodb]
command=/data/mongodb/bin/mongod --dbpath /data/database/mongodb
directory=/data/mongodb/bin/
autorestart=true
redirect_stderr=true

[program:leanote]
command=bash /data/leanote/bin/run.sh
directory=/data/leanote/bin
autorestart=true
redirect_stderr=true
# 启动
supervisord -c /etc/supervisord.conf

绑定域名和端口

vim /data/leanote/conf/app.conf
http.port=61885
site.url=http://192.168.1.126:61885

nginx 转发

server{
        listen 80;
        server_name leanote.seacme.cn;
        error_log  /data/logs/nginx/leanotes_error.log;

        location / {
            proxy_pass http://192.168.1.126:61885;
            proxy_read_timeout 300;
            proxy_connect_timeout 300;
            proxy_redirect     off;

            proxy_set_header   X-Forwarded-Proto $scheme;
            proxy_set_header   Host              $http_host;
            proxy_set_header   X-Real-IP         $remote_addr;
        }
}
