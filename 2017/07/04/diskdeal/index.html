2T 以上磁盘格式化

Jul 4, 2017 #Linux

利用parted工具对磁盘格式化操作

一、安装parted工具

yum -y install parted

二、用parted工具分区磁盘

parted -s /dev/sdb "mklabel gpt"
# 分区模式 gpt
parted -s /dev/sdb "mkpart primary 0% 100%"
# 将整块磁盘进行分区

三、格式化磁盘

mkfs.ext4 /dev/sdb1

或者采用：

mkfs.ext4 -E lazy_itable_init=1 /dev/sdb1
# 延迟分配inode，挂载后再分配，加快mkfs速度

四、修改磁盘卷标、修改自动挂载并挂载

# 修改磁盘卷标
e2label /dev/sdb1 /data
blkid /dev/sdb1
# 获取磁盘UUID信息
vim /etc/fstab
UUID=54dea55f-da4e-429e-8ae3-34eecfc00448 /data	                  ext4    defaults,noatime       1 3
# 挂载磁盘到指定目录
mount /dev/sdb1 /data

五、检查磁盘情况

df -hT
