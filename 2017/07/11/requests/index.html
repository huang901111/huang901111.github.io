Python Requests模块

Jul 11, 2017 #python

Python爬虫利器一之Requests库

一、Requests模块

Reuqests是Python的一个HTTP客户端库，跟urllib、urllib2类似。相对urllib之类的模块，对比下发现它确实很人性化。是一个『人类』用的http模块。

二、Requests基础使用

2.1 发送请求操作

import requests
r = requests.get('http://www.baidu.com')
r = requests.post('http://www.baidu.com/post')
r = requests.put('http://www.baidu.com/put')
r = requests.delete('http://www.baidu.com/delete')
r = requests.head('http://www.baidu.com/get')
r = requests.options('http://www.baidu.com/get')

2.2 URL中传递参数

payload = {'wd':'xxx','rn':'yyy'}
r = requests.get('http://www.baidu.com/s',params=payload)
print r.url
u'http://www.baidu.com/s?rn=yyy&wd=xxx'

2.3 获取响应内容

  • 通过r.text来获取网页内容（返回unicode字体）
r = requests.get('http://www.baidu.com')
r.text
u'<!DOCTYPE html>\r\n<!--STATUS OK--><html> <head><meta http-equiv=content-type content=text/html;charset=utf-8><meta http-equiv=X-UA-Compatible content=IE=Edge>
  • 通过r.content来获取页面内容（返回二进制字节方式）
r = requests.get('http://www.baidu.com')
r.content
'<!DOCTYPE html>\r\n<!--STATUS OK--><html> <head><meta http-equiv=content-type content=text/html;charset=utf-8><meta http-equiv=X-UA-Compatible content=IE=Edge>

2.4 获取网页编码

  • 通过r.encoding来获取网页编码
r = requests.get('http://www.baidu.com')
r.encoding
'ISO-8859-1'

2.5 json

  • 像urllib和urllib2，如果用到json，就要引入新模块，如json和simplejson，但是在requets中已经有了内置函数r.json()
r = requests.get('http://ip.taobao.com/service/getIpInfo.php?ip=122.88.60.28')
r.json()['data']['country']
'中国'

2.6 网页状态码

  • 通过r.status_code来检查网页的状态码
r = requests.get('http://www.baidu.com')
r.status_code
200

r = requests.get('http://www.mengtiankong.com/12121')
r.status_code
404

r = requests.get('http://www.baidu.com/link?url=QeTRFOS7TuUQRppa0wlTJJr6FfIYI1DJprJukx4Qy0XnsDO_s9baoO8u1wvjxgqN')
# 实际为百度的跳转链接
r.url
u'http://www.zhidaow.com'
# 实际跳转到www.zhidaow.com
r.status_code
200
# 实际上跳转返回的状态码为302才是正确的
r.history
(<Response [302]>,)

# 禁止跳转
r = requests.get('http://www.baidu.com/link?url=QeTRFOS7TuUQRppa0wlTJJr6FfIYI1DJprJukx4Qy0XnsDO_s9baoO8u1wvjxgqN',allow_redirects = False)
r.status_code
302

2.7 获取响应头内容

  • 通过r.headers来获取响应头内容
r = requests.get('http://www.zhidaow.com')
r.headers
{
  'x-cache-key': 'f1882365ff3666cee5e1e2d0803f12e9-page', 
  'x-template-path': 'index+tags+category.jade',
......
}

# 获取响应头的部分内容
r.headers['Content-Type']
'text/html'

r.headers['x-cache-key']
'f1882365ff3666cee5e1e2d0803f12e9-page'

2.8 设置超时时间

  • 通过关键字timeout
r = requests.get('http://github.com',timeout=0.0.1)
HTTPConnectionPool(host='github.com', port=80): Max retries exceeded with url: / (Caused by ConnectTimeoutError(<requests.packages.urllib3.connection.HTTPConnection object at 0x303a610>, 'Connection to github.com timed out. (connect timeout=0.01)'))

2.9 代理访问

  • 采集时为了避免被封IP，经常会使用代理。requests也有相应proxies属性
import requests

proxies = {

  "http":"http://10.10.1.10:3128",
  "https":"http://10.10.1.10:1000",
}

requests.get("http://www.zhidaow.com",proxies=proxies)

# 若代理需要账户和密码

proxies = {

    "http":"http://user:pass@10.10.1.10:3123"
  
}

2.10 获取请求头内容

  • 请求头内容可以用r.request.headers来获取
r = requests.get('http://www.zhidaow.com')
r.request.headers
 {'Connection': 'keep-alive', 'Accept-Encoding': 'gzip, deflate', 'Accept': '*/*', 'User-Agent': 'python-requests/2.12.3'}

2.11 自定义请求头部

  • 伪装请求头部是采集时经常用的，通过这个方法来隐藏
r = requests.get('http://www.zhidaow.com')
print r.request.headers['User-Agent']

headers = {'User-Agent':'alexkh'}
r = requests.get('http://www.zhidaow.com',headers=headers)
print r.requests.headers

2.12 持久连接keep-alive

  • requests的keep-alive是基于urllib3，同一会话内的持久连接完全是自动的。同一会话内的所有请求都会自动使用恰当的连接
  • 无需任何设置，requests会自动实现keep-alive

三、Requests 高级应用

3.0 简单的Get与Post请求

  • 无参数的get请求
import requests

r = requests.get('http://www.baidu.com',timeout=1) # 设置超时，超时候会抛出timeout异常
print r.text # 一般用来输出纯文本，可得到unicode类型字符串
print r.content # 一般用来输出pdf、图片等，可得到原网页设定类型的字符串
  • 有参数的get请求
import requests
url = 'http://10.10.10.10:8080/Lab2.0/Login.action'
header = {

  'User-Agent':'Mozilla/5.0 (Windows NT 10.0; Win64; x64; rv:47.0) Gecko/20100101 Firefox/47.0',

}

payload = {'aaa':'1111','bbb':'2222'}
r = requests.get(url,params=payload,headers=header)
print r.url # 得到url
print r.status_code # 得到返回的状态码
print r.headers # 得到html头
  • 有参数的post请求
import requests

url1 = 'http://10.10.10.10:8080/Lab2.0/Login.action'
url2 = 'http://10.10.10.10:8080/Lab2.0/student.action'

payload = {

  'userid':'xxxxx',
  'password':'xxxxxx',
  'quan':'xxxxx',

}

  'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64; rv:47.0) Gecko/20100101 Firefox/47.0',
  'Content-Type': 'application/x-www-form-urlencoded; charset=UTF-8',

}

r = requests.post(url,data=payload,headers=header)
cookies = r.cookies # 保存cookies

3.1 淘宝IP查询服务

import requests

url = 'http://ip.taobao.com/service/getIpInfo.php'
payload = {'IP':'125.90.88.48'}
r = requests.get(url,params = payload)
print r.json()['data']['country_id']

3.2 定制header模拟

import requests
import json

data = {'some':'data'}
headers = {

  'content-type':'application/json'
  'User-Agent':'Mozalla/5.0 (X11; Ubuntu; Linux x86_64; rv:22.0) Gecko/20100101 Firefox/22.0'
  
}

r = requests.post('http://api.github.com/some/endpoint', data=data, headers=headers)
print r.text

3.3 上传与下载文件

  • 使用Request模块进行文件上传，文件的类型会自动进行处理
import requests

url = 'http://127.0.0.1:8000/upload'
file = {'myfile':open('/home/lyb/aaa.jpg','rb'),'bb.jpg'}
r = requests.post(url,files=file)
print r.text
  • 使用Requests模块进行文件下载
import urllib
import requests

# 下载不同类型的文件只要更改后缀即可

url = 'http://nginx.org/download/nginx-1.9.9.tar.gz'

r = requests.get(url)
with open('a.tar.gz','wb') as f:
  f.write(r.content)
  
urllib.urlretrieve(url,'b.tar.gz')

3.4 身份验证

  • 基本身份认证（HTTP Basic Auth）
import requests
from requests.auth import HTTPBasicAuth

r = requests.get('http://httpbin.org/hidden-basic-auth/user/passwd', auth=HTTPBasicAuth('user', 'passwd'))

print r.json

3.5 Cookies 与 Session

  • 如果某个响应中包含一些Cookies，可以快速访问他们
import requests

r = requests.get('http://www.google.com.hk')
print r.cookies['IND']
print tuple(r.cookies)
  • 发送cookies到服务器
import requests

url = 'http://httpbin.org/cookies'
cookies = {'testCookies_1': 'Hello_Python3', 'testCookies_2': 'Hello_Requests'}
# 在Cookie Version 0中规定空格、方括号、圆括号、等于号、逗号、双引号、斜杠、问号、@，冒号，分号等特殊符号都不能作为Cookie的内容。
r = requests.get(url, cookies=cookies)
print r.json()
  • Session能够跨请求保持某系参数，在同一个Session实例发出所有请求之间保持cookies，且这些都是自动处理
import requests

url = 'http://10.10.10.10:8080/Lab2.0/Login.action'

proxy = {

  'http':'http://127.0.0.1:8000'

}

payload = {

  'userid':'xxxx',
  'password':'xxxxx',
  'quan':'xxxxx',

}

s = requests.Session() # 此后请求时不用再声明cookies
r = s.post(url,data=payload,proxies=proxy)
# 此时再次请求就不用使用cookie了
r = s.get('http://10.10.10.10:8080/Lab2.0/student.action')
print r.text
