Python Urllib模块

Jul 12, 2017 #python

Python爬虫利器二之Urllib/Urllib2库

一、urllib与urllib2认识

Python的urllib和urllib2模块都做与请求url相关的操作，但他们提供不同的功能。

  • urllib2可以接受一个Request对象，并以此可以设置一个URL的headers，但是urllib只接受一个URL。这意味着，不能伪装你的用户代理字符串等。
  • urllib模块可以提供进行urlencode的方法，该方法用于GET查询字符串的生成，urllib2的不具有这样的功能，这就是urllib与urllib2经常一起使用的原因。
  • 可以说urllib2是对urllib的增强。

二、urllib 方法介绍

2.1 urlopen

  • 介绍

    • 创建一个表示远程url的类文件对象，然后像本地文件一样操作这个类文件对象来获取远程数据。

  • 用法

    urllib.urlopen(url[, data[, proxies]])

  • 参数说明

    • url 表示远程数据的路径，一般是网址
    • data 表示以post/get方式提交到url的数据
    • proxies 用于设置代理

  • 返回

    • read()、readlines()、fileno()、close()，这些方法的使用法与文件对象完全一样
    • info() 返回一个httplib.HTTPMessage对象，表示远程服务器返回的头信息
    • getcode() 返回http状态码
    • geturl() 返回请求的url

2.2 urlretireve

  • 介绍

    • 直接在远程数据下载到本地，该方法返回一个包含两个元素的元祖（filename,headers）
    • filename 表示保存到本地的路径
    • header 表示服务器的响应头

  • 用法

  urllib.urlretrieve(url[,filename[,reporthook[,data]]])

  • 参数说明

    • url 外部或者本地url
    • filename 指定了保存到本地的路径
    • reporthook 是一个回调函数，当连接上服务器，以及相应的数据块传输完毕的时候会触发该回调，可以利用这个回调函数来显示当前的下载进度。
    • data 指post/get到服务器的数据。

2.3 urlcleanup

  • 介绍

    • 清除由于urllib.urlretrieve()所产生的缓存

2.4 quote、quote_plus

  • 介绍

    • 将url数据获取之后，并将其编码，从而适用与URL字符串中，使其能被打印和被web服务器接受。

  • 两者差异

    • quote_plus()方法与quote()方法类似，但quote_plus方法是用”+“来替换” “

2.5 unquote、unquote_plus

  • 介绍

    • 对字符串进行解码

2.6 urlencode

  • 介绍

    • 将dict或者包含两个元素的元祖列表转换成url参数。
    • 例如：dict{‘name’:‘dark-bull’,‘age’:‘20’} 将被转换为name=dark-bull&age=20

2.7 pathname2url

  • 介绍

    • 将本地路径转换成url路径

2.8 实用例子

  #!/usr/bin/env python
  # -*- coding:utf-8 -*-

  import urllib

  s = urllib.quote('This is Python') # 编码
  print 'quote:\t' + s
  s_un = urllib.unquote(s)
  print 'unquote:\t' + s_un
  s_plus = urllib.quote_plus('This is Python') # 编码
  print 'quote_plus:\t' + s_plus
  s_unplus = urllib.unquote_plus(s_plus)
  print 'unquote_plus:\t' + s_unplus

  # 编码字典转换为url参数
  s_dict = {'name':'tq','pass':'1234'}
  s_encode = urllib.urlencode(s_dict)
  print 's_encode:\t' + s_encode

三、urllib2 方法介绍

3.1 urlopen

用法与urllib里的urlopen方法类似，区别urllib2.urlopen()可以打开一个Request对象，可选参数timeout，阻塞操作以秒为单位

  • 最简单用法
  import urllib2

  url = urllib.urlopen(url)
  response = urllib2.urlopen(url)
  page = response.read()

  • 日常用法

    • urliopen通过建立一个Request对象来明确指明想要获取的url，调用urlopen函数对请求的url返回一个response对象，这个response类似于一个file对象，所有用.read()函数可以操作这个response对象
  import urllib2

  url = 'http://www.baidu.com'

  request = urllib2.Request(url)
  response = urllib2.urlopen(request)
  page = response.read()
  • Requests 类是一个抽象的URL请求（传参用法）
  import urllib2
  import urllib

  url = 'http://www.xxxx.com/cgi-bin/register.cgi'
  values = {

  • 设置请求超时

    • 好多时候各种原因，有可能导致你的请求各种等待。

    • 这个时候可通过设置urlopen中的超时去掉那些无法容忍的响应请求

    • 使用timeout的时候要注意一点，如果没有data数据，那么这时一定要显示的传递参数

  import urllib2
  urllib2.urlopen('http://www.baidu.com',data,10)
  urllib2.urlopen('http://www.baidu.com',timeout=10)

  • headers(设置HTTP请求头)

    • 有一些网站不希望被某些程序浏览或者针对不同的浏览器返回不同的版本。

  • install_opener、build_opener

    • install_opener和build_opener这两个方法通常都是一起用，也有时候build_opener单独使用来得到OpenerDirector对象。
    • install_opener 实例化会得到OpenerDirector对象用来赋予全局变量opener。
    • build_opener实例化也会得到OpenerDirector对象，其中参数handlers可以被BaseHandler或它的子类实例化。
    • 子类可以通过以下实例化：ProxyHandler（代理使用）、UnknownHnadler、HTTPHandler、HTTPDefaultErrorHandler, HTTPRedirectHandler, FTPHandler, FileHandler, HTTPErrorProcessor。

  • 设置代理

  import urllib2 
  proxy_handler = urllib2.ProxyHandler({"http":'http://11.11.11.11:8080'})
  opener = urllib2.build_opener(proxy_handler)
  urllib2.install_opener(opener)
  • 设置http和https的Debug log功能
  import urllib2
  httpHandler = urllib2.HTTPHandler('debuglevel=1')
  httpsHandler = urllib2.HTTPSHandler('debuglevel=1')
  opener = urllib2.build_opener(httpHandler, httpsHandler)
  urllib2.install_opener(opener)
  response = urllib2.urlopen('http://www.baidu.com')
  • 结合cookielib处理cookie信息
  import urllib2
  import cookielib

  cookie = cookielib.CookieJar()
  cookieHandler = urllib2.HTTPCookieProcessor(cookie)
  opener = urllib2.build_opener(cookieHandler)
  urllib2.install_opener(opener)
  response = urllib2.urlopen('http://www.baidu.com')
  for item in cookie:
      print 'CookieName = '+item.name
      print 'CookieValue = '+item.value

四、urllib异常处理

  from urllib2 import Request, urlopen, URLError  
  req = Request(someurl)  
  try:  
    response = urlopen(req)  
  except URLError, e:  
    if hasattr(e, 'reason'):  
      print 'We failed to reach a server.'  
      print 'Reason: ', e.reason  
    elif hasattr(e, 'code'):  
      print 'The server couldn/'t fulfill the request.'  
      print 'Error code: ', e.code  
    else:
      pass
