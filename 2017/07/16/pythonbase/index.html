1.1 List.append 方法

Lambda 函数

2.1 Lambda 函数介绍

  • Python 中使用 Lambda 来创建匿名函数
  • Lambda 的主体是一个表达式，而不是一个代码块，仅仅能在 Lambda 表达式中封装有限的逻辑
  • Lambda 函数拥有自己的命名空间，且不能访问自有参数列表之外或全局命名空间里的参数

2.2 Lambda 优势

  • 使用 Lambda 可以省去定义函数的过程，让代码更加精简
  • 对于一些抽象的，不会在别的地方再复用的函数
  • 使用 Lambda 能让代码更容易理解

2.3 Lamdba 基础使用

  • 声明 sum 函数求和
  def sum(x,y):
  	return x+y
  sum(10,20)

  ### 使用 Lambdb 让代码更加简洁
  sum = lambda x,y:x+y
  sum
  <function __main__.<lambda>>
  ### sum 作为函数表达式使用
  sum(10,20)
  或： 
  sum = (lambda x,y:x+y)(10,20)

Map 函数

  • 大多数的for循环可以用map来代替
  • map(func,seq) 自动地将函数表达式（func）施加到 seq（可迭代对象）的每一个元素对象，并返回一个列表
  map(...)
      map(function, sequence[, sequence, ...]) -> list

Filter 函数

  • Filter 是对一个 seq（可迭代对象）中的每一个元素进行过滤，返回符合条件的那些元素

  • Filter(fund,seq) 自动地将函数表达式（func）施加到 seq（可迭代对象）的每一个元素对象，并返回符合条件的列表、元组或者字符串。

  filter(...)
      filter(function or None, sequence) -> list, tuple, or string

4.1 Filter 函数实例

  • 获取 List 中的所有奇数
  List = [3,5,7,12,18]

  ### Filter函数配合Lambda函数
  JiShu = filter(lambda x:x%2,List)
  print JiShu
  [3, 5, 7]

  ### 列表解释器
  JiShu = [ i for i in List if i%2]
  print JiShu
  [3, 5, 7]
  • 获取列表中出现匹配的某字符串
  sentence = ['Hi Steven!','Hi David','How are you?']

  ### Filter函数配合Lambda函数
  partern = filter(lambda x:x.count('Hi'),sentence)
  print partern
  ['Hi Steven!', 'Hi David']
  ### 返回的并不是“Hi“出现的次数，而是返回存在“Hi”的对象元素

  ### 列表解释器
  partern = [ i for i in sentence if i.count('Hi')]
  print partern
  ['Hi Steven!', 'Hi David']
  • 获取列表值中符合条件的字符串
  games = ['xxx_xxx_1220','xxx_xxx_9999','xxx_xxx_8999']

  ### Filter函数配合Lambda函数
  sids = filter(lambda x:int(x.split('_')[-1])>8000,games)
  print sids
  ['xxx_xxx_9999', 'xxx_xxx_8999']
  ### 获取sid值比8000大的值
  • 误区：
  sids = filter(lambda x:x.split('_')[-1]>8000,games)
  print sids
  ['xxx_xxx_1220', 'xxx_xxx_9999', 'xxx_xxx_8999']
  ### 由于列表元素截取到的是字符串，字符串与整数型无法比较，导致出错

Reduce 函数

  • Reduce(func,seq) 是对seq（可迭代对象）中的每个元素进行func操作，每次携带一对（先前的结果以及下一个序列的元素），连续的将现有的结果和下一个值作用在获得的随后的结果上，最后减少我们的序列为一个单一的返回值。

  • Reduce(func,seq,starting_value) 如果有starting_value，还可以作为初始值调用

  reduce(...)
      reduce(function, sequence[, initial]) -> value

5.1 Reduce 函数实例

  • 求 List 列表中所有元素的乘积
  List = [1,2,3,4,5]

  ### Reduce函数配合Lambda函数
  chengji = reduce(lambda x,y:x*y,List)
  24
  ### 此处lambad函数有两个参数x和y，reduce会将List里面的头两个参数作为x和y，求他们的乘积，然后再把结果与第3个数相乘，以此类推，最后输出乘积结果
  • 获取 List 列表中的最大值
  List = [1,2,3,4]

  ### Reduce函数配合Lambda函数
  max = reduce(lambda x,y:x if x>y else y,List)
  4
  ### 此处lambda函数有两个参数x和y，reduce会将List里面的头两个参数进行比较，获取前两者比较大值与第3个值进行比较，以此类推，最后输出最大值
  或
  max = reduce(max,List)
  4
  • 获取 List 列表中的平方和
  List = [1,2,3,4]

  ### Reduce函数配合Lambda函数
  sum = reduce(lambda x,y:x+y**2,List,0)
  print sum
  • 误区： 
  sum = reduce(lambda x,y:x**2+y**2,List)
  ### 以上操作会获取到错误的结果，由于平方和的实现形式是列表中数值平方相加
  ### 以上操作是抽取列表前2个数值进行平方求和，获取到结果平方与第3个值平方相加，以此类推，得出错误的结果
  • 列表中的元素为列表，将其组合成1个列表
  a = [[1,2],[3,4],[5,6]]

  ### Reduce函数配合Lambda函数
  b = reduce(lambda x,y:x+y,a)
  print b
  [1, 2, 3, 4, 5, 6]
  • 统计某字符串出现次数
  sentences = ['Mary read a story to Sam and Isla.',
               'Isla cuddled Sam.',
               'Sam chortled.']
               
  ### Reduce函数配合Lambda函数
  count = reduce(lambda a,x:a+x.count('Sam'),sentences,0)
  print count
  3

Zip 函数

7.1 Zip 函数介绍

  • Zip() 是Python 中的一个内建函数，它接受一系列可迭代的对象作为参数，将对象中的元素打包成一个个tuple（元组），然后返回由这些tuples组成的list（列表）。
  • 注意：Zip()函数生成新的序列的长度以参数中最短的序列为准。

7.2 Zip 函数实例

  • 新的序列的长度以参数中最短的序列为准
  a = ['1','2']
  b = ['x','y','z']
  zip(a,b)
  [('1', 'x'), ('2', 'y')]
  • (*)操作符与zip函数配合可以实现与zip相反的功能，即将合并的序列拆分为多个tuple（元组）
  a = ['1','2','3']
  b = ['x','y','z']
  zip(*zip(a,b))
  [('1', '2', '3'), ('x', 'y', 'z')]
  • 两个清单组成购物列表
  products = ['Apple','Orange','Banana']
  prices = [14,15,16]

  ### 利用Zip函数组成购物列表
  ShopList = zip(products,prices)
  print ShopList
  [('Apple', 14), ('Orange', 15), ('Banana', 16)]
  • 二维矩阵变换（矩阵的行列互换）
  a = [[1,2,3],[4,5,6],[7,8,9]]

  ### 利用Zip函数将矩阵的行列互换
  b = zip(*a)
  print b 
  [(1, 4, 7), (2, 5, 8), (3, 6, 9)]

  b = map(list,zip(*a))
  print b
  [[1, 4, 7], [2, 5, 8], [3, 6, 9]]

  ### 利用列表解释器
  b = [[ row[col] for row in a ] for col in range(len(a[0]))]
  print b
  [[1, 4, 7], [2, 5, 8], [3, 6, 9]]
  • 合并列表相邻的列表项
  a = [1,2,3,4,5,6,7]
  zip(*([iter(a)] * 2))
  [(1, 2), (3, 4), (5, 6)]
  ### 以上按每2个元素组成tuple（元组），由多个tuples组成列表
  zip(*([iter(a)] * 3))
  [(1, 2, 3), (4, 5, 6)]
  ### 以上按每3个元素组成tuple（元组），由多个tuples组成列表
  • 使用Zip()函数反转字典
  m = {'a': 1, 'b': 2, 'c': 3, 'd': 4}
  zip(m.values(),m.keys())
  [(1, 'a'), (3, 'c'), (2, 'b'), (4, 'd')]
  m_new = dict(zip(m.values(),m.keys()))
  print m_new
  {1: 'a', 2: 'b', 3: 'c', 4: 'd'}

Any All 函数

  • all ：当 seq （可迭代对象）为空（empty）或 seq （可迭代对象）中的所有元素都为真（True）时则返回True，否则返回False。
  all(...)
      all(iterable) -> bool
      Return True if bool(x) is True for all values x in the iterable.
  • any : 当 seq （可迭代对象）不为空且任意一个元素为真（True）则返回True，否则返回False。
  any(...)
      any(iterable) -> bool 
      Return True if bool(x) is True for any x in the iterable.
  • 此两个函数的参数都是 iterable（可迭代对象），也就是 list 、tuple 或者 dict。

8.1 Any All 函数实例

  • All() 函数简单使用
  all(['a',(2,4),1,True])
  True
  all(['a',(),1,True])
  False
  ### 由于列表中存在空的tuple元组
  all(['a',(2,4),0,True])
  False
  ### 由于列表中存在0值
  all(['a',(2,4),3,False])
  False
  ### 由于列表中存在false值
  all([])
  True
  all(())
  True
  all({})
  True
  all('')
  True
  ### 以上证明可迭代对象（字符串、列表、元组和字典）为空，返回结果都为True
  • Any() 函数简单使用
  any(['a',(2,4),3,True])
  True

Dict（字典）

12.1 Dict（字典）介绍

  • 字典 是另外一种可变容器模型，且可存储任意类型对象。
  • 字典 由键（key）和对应值（value）成对组成。字典 也称做关联数组或哈希表。

12.2 Dict（字典）特征

  • Dict（字典）中的key（键）必须独一无二，但是对应值（value）则不必。
  • Dict（字典）不允许同一个键出现2次，创建时如果同一个键被赋值两次，后一个值会被记住。
  • key需要可以被哈希化，因此不能是可变型对象。
  • value可以是任何东西，但必须是不可变的，包括字符串、数、列表、元组或者字典等。
  • 空字典可以随意用赋值添加项，而列表不行。如 List = [],List[1]=‘Yes’ 将会出现报错，而D1 = {},D1{‘A’}=100则正常。
