Python Glob模块

Jul 28, 2017 #Python

Glob模块可用于根据特定规则遍历搜索文件路径名

一、glob模块介绍

  1. glob 的应用场景是要寻找一系列（符合特定规则）文件名

  2. glob 模块是最简单的模块之一，内容非常少。用它可以查找符合特定规则的文件路径名。查找文件只用到三个匹配符：”*“、”?“、”[]”

    • * 匹配0个或多个字符
    • ? 匹配单个字符
    • [] 匹配指定范围内的字符，如：[0-9]匹配数字。

二、glob匹配

2.1 匹配所有文件

  • 可以用 * 匹配任意长度字节。glob.glob比较常用，返回一个list。也可用glob.iglob返回生成器。
import glob
for name in glob.glob('dir/*'):
    print name 
# 找到dir目录下的所有文件

2.2 匹配子目录文件

  • 可以指定子目录名称，也可以通配符代替，不显示指定。
import glob
for name in glob.glob('dir/subdir/*'):
    print '\t',name
    
for name in glob.glob('dir/*/*'):
    print '\t',name

  • 单字节通配符匹配

    • 除了 * 以外，还有 ? 匹配单个字符
    import glob
for name in glob.glob('dir/file?.txt'):
    print name

  • 字符区间匹配[0-9]

    • 比如匹配后缀前是数字的文件
    for name in glob.glob('dir/*[0-9].*'):
    print name

补充：

  1. glob.glob0(dir,path)：要是存在dir+os.sep+path就返回[path].基础判断函数，用来加法
  2. glob.glob1(dir,pattern)：就是返回目录下匹配模式的，glob主要靠他。
  3. glob.has_magic(s)：判断字符串里是否有通配符，有就返回True
  4. glob.magic_check：一个re的正则表达式对象
  5. glob.re/sys/os/fnmatch：用到的模块
