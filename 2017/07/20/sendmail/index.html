Linux 发送邮件

Jul 20, 2017 #Linux

三种发送邮件方式，任君选择

【方法1】 利用自带sendmail服务

1.1 修改/etc/mail.rc配置文件

vim /etc/mail.rc
set smtp-use-starttls
set from="xxxxx@xxxxx.com"
# 设置邮件发件人名称
set smtp="xxxx.xxxxx.com:xxx"
# 设置发送邮件的smtp服务器和端口
set smtp-auth-user="xxxxxxxx"
# 设置SMTP服务器认证用户
set smtp-auth-password="xxxxxxxx"
# 设置SMTP服务器认证密码
set smtp-auth=login
set nss-config-dir=/etc/pki/nssdb/
set ssl-verify=ignore

1.2 重启sendmail服务并测试

/etc/init.d/sendmail start
mail -s "Test_SendMail"  ${receiver} < test.txt

【方法2】 利用ssmtp服务

2.1 安装部署ssmtp服务

wget http://download.fedoraproject.org/pub/epel/6/i386/epel-release-6-8.noarch.rpm
# 利用epel源安装ssmtp服务
yum -y install ssmtp

2.2 修改/etc/ssmtp/ssmtp.conf配置

echo " " > /etc/ssmtp/ssmtp.conf
cat >/etc/ssmtp/ssmtp.conf<<EOF
root=xxxx@xxxxx.com
# 设置邮件发件人名称
mailhub=xxxx.xxxxx.com:xxx
# 设置发送邮件服务器和端口
UseSTARTTLS=YES
Hostname=xxxx.xxxx.com
# 设置邮件服务器主机名
AuthUser=xxxx
# 设置登录邮件服务器账户
AuthPass=xxxxxxxx
# 设置登录邮件服务器密码
FromLineOverride=YES
EOF

2.3 发送邮件测试

mail -s "Test_SendMail"  ${receiver} < test.txt

【方法3】 利用Msmtp和Mutt发送邮件

3.1 下载Msmtp和Mutt软件安装包

  • Msmtp Download
  • Mutt Download

3.2 安装部署 Msmtp

cd /data/backup/tmp
tar -Jxf msmtp-1.6.4.tar.xz
cd msmtp-1.6.4
./configure --prefix=/usr/local/msmtp 
make && make install
ln -s /usr/local/msmtp/bin/msmtp /usr/bin/msmtp

vim /root/.msmtprc
account mingchao
host mail.xxxxxx.com
port xxxx
# 设置发送邮件服务器和端口
from xxxxxx@mingchao.com
# 设置发送邮件人
auth login
user xxxxxx
password xxxxxxx
tls on
tls_starttls on
tls_certcheck off
# 由于邮件服务器开启了tls必须要加以上3项
account default:mingchao
# 设置默认发送邮件邮箱
logfile ~/.msmtp.log
chmod 600 /root/.msmtprc
touch ~/.msmtp.log

3.3 查看邮件服务器是否支持tls认证加密

msmtp --host=mail.xxxx.com --serverinfo
SMTP server at mail.mingchao.com ([xxxxxxxx]), port xxxxx:
    mail.mingchaoonline.com ESMTP Postfix
Capabilities:
    SIZE 512000000:
        Maximum message size is 512000000 bytes = 488.28 MiB
    PIPELINING:
        Support for command grouping for faster transmission
    ETRN:
        Support for RMQS (Remote Message Queue Starting)
    DSN:
        Support for Delivery Status Notifications
    STARTTLS:
        Support for TLS encryption via the STARTTLS command
This server might advertise more or other capabilities when TLS is active.

3.4 利用msmtp发邮件测试

msmtp xxxxx@mingchao.com
Test
ctrl+d

3.5 安装部署 Mutt

cd /data/backup/tmp/
tar -zxf mutt-1.6.0.tar.gz
cd mutt-1.6.0
./configure --prefix=/usr/local/mutt
make && make install
ln -s /usr/local/mutt/bin/mutt /usr/bin/mutt

vim /root/.muttrc
set sendmail="/usr/bin/msmtp"
set from="xxxxxxx@mingchao.com"
# 设置邮件发送人信息
chmod 600 /root/.muttrc

3.6 利用Msmtp+Mutt测试发送邮件

echo "Test" | mutt xxxx@mingchao.com -s "Test20160427" -a /data/tmp/test_sendmail.txt
# -a 在邮件中加入附件
# -i 将指定文件插入邮件内文中

【注意】

  • Centos 5 sendmail 只允许使用本地的smtp服务器，本机需要开启sendmail或者poftfix，可以借助ssmtp或者升级mailx到12.4版本以上
  • Centos 6 可以使用外部smtp服务器
