SmokePing 部署

Jul 25, 2017 #Network

一款用于监控IDC多线路网络质量工具

一、SmokePing 简介

  • smokeping是rrdtool的作者Tobi Oetiker的作品，所以它在图形显示方面有很大优势，也是一个很有特点的opensource工具;
  • 多种探测方式，包括fping、echoping、dig、curl等；
  • 易用可扩展的插件；
  • master/slave的工作方式，可以在多个节点收集同一个监测点的数据；很有特色的alert设置，不只是简单的设置一个阀值；

二、SmokePing 安装部署

2.1 环境准备

### 安装epel源
rpm -Uvh http://apt.sw.be/redhat/el6/en/x86_64/rpmforge/RPMS/rpmforge-release-0.5.3-1.el6.rf.x86_64.rpm
### 安装服务、依赖包
yum install -y httpd httpd-devel  rrdtool-perl popt-devel libidn-devel perl-ExtUtils-Embed

2.2 软件源码包下载

### fping源码包
wget http://www.fping.org/dist/fping-3.10.tar.gz
### echoping源码包
wget http://nchc.dl.sourceforge.net/project/echoping/echoping/6.0.2/echoping-6.0.2.tar.gz
### smokeping源码包
wget http://oss.oetiker.ch/smokeping/pub/smokeping-2.6.8.tar.gz</pre>

2.3 Fping 源码部署

tar -zxvf fping-3.10.tar.gz
cd fping-3.10
./configure && make && make install

2.4 Echoping 源码部署

tar -zxvf echoping-6.0.2.tar.gz
cd echoping-6.0.2
./configure && make && make install

2.5 SmokePing 源码部署

tar -zxvf smokeping-2.6.8.tar.gz 
cd smokeping-2.6.8
./setup/build-perl-modules.sh  /usr/local/smokeping/thirdparty
### 由于smokeping默认使用很多perl扩展包，推荐以上方式一次性安装
./configure  --prefix=/usr/local/smokeping
/usr/bin/gmake install

2.6 建立 SmokePing 工作目录，并赋权

mkdir /usr/local/smokeping/{data,cache,var,logs}
chown -R apache.apache /usr/local/smokeping
chmod 600 /usr/local/smokeping/etc/smokeping_secrets.dist
chmod 755 -R /usr/local/smokeping/{data,var,cache,logs}

2.7 初始化并配置 SmokePing 相关文件

mv /usr/local/smokeping/htdocs/smokeping.fcgi.dist /usr/local/smokeping/htdocs/smokeping.fcgi
cp /usr/local/smokeping/etc/config.dist /usr/local/smokeping/etc/config
vim /usr/local/smokeping/etc/config
imgcache = /usr/local/smokeping/cache
datadir  = /usr/local/smokeping/data
piddir  = /usr/local/smokeping/var
cgiurl   = http://192.168.50.150/smokeping.cgi
binary = /usr/local/sbin/fping
### 确定以上配置的正确
touch /usr/local/smokeping/logs/smokeping.log

2.8 修改httpd.conf加入 SmokePing 虚拟主机目录

vim /etc/httpd/conf/httpd.conf
Alias /smokeping  "/usr/local/smokeping/htdocs"
Alias /cache  "/usr/local/smokeping/cache"
Alias /cropper  "/usr/local/smokeping/htdocs/cropper"

<Directory "/usr/local/smokeping/htdocs">
     AllowOverride None
     AddHandler cgi-script .fcgi .cgi
     Options ExecCGI
     <IfModule dir_module>
          DirectoryIndex smokeping.fcgi
     </IfModule>
     Order allow,deny
    Allow from all
</Directory>

三、启动 SmokePing 服务

3.1 SmokePing 启动脚本

#!/bin/bash
. /etc/init.d/functions
# Start the service Smokeping
start() {
echo -n "Starting Smokeping: "
/usr/local/smokeping/bin/smokeping --logfile=/usr/local/smokeping/logs/smokeping.log >
 /dev/null 2>&1
### Create the lock file ###
touch /usr/local/smokeping/smokeping.pid
success $"Smokeping startup"
echo
}

# Restart the service Smokeping
stop() {
echo -n "Stoping Smokeping: "
kill -9 `ps ax | grep "/usr/local/smokeping/bin/smokeping" | grep -v grep | awk '{ pri
nt $1 }'` > /dev/null 2>&1
### Now, delete the lock file ###
rm -f /usr/local/smokeping/smokeping.pid
success $"Smokeping shutdown"
echo
}

### main logic ###
case "$1" in
start)
start
;;
stop)
stop
;;
status)
status Smokeping
;;
restart|reload|condrestart)
stop
start
;;
*)
echo $"Usage: $0 {start|stop|restart|reload|status}"
exit 1
esac
exit 0

3.2 设置 SmokePing 启动脚本

chmod +x /etc/rc.d/init.d/smokeping
chkconfig --add smokeping
chkconfig smokeping on
