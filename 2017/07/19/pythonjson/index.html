Python JSON模块

Jul 19, 2017 #Python

一个处理轻量级的数据交换格式JSON模块

一、JSON 概述

1.1 介绍

  • JSON(JavaScript Object Notation)是一种轻量级的数据交换格式。易于人阅读和编写。同时也易于机器解析和生成。
  • JSON采用完全独立于语言的文本格式，但是也使用了类似C语言家族的习惯。

1.2 JSON建构两种结构

  • JSON只提供了两种结构：字典 和 数组
  • 名称/值对的集合（A collection of name/value pairs）。在不同的语言中，它被理解为对象（object）、纪录（record）、结构（struct）、字典（dictionary）、哈希表（hash table）、有键列表（keyed list）或者关联数组（associative array）。—-> 字典
  • 值得有序列表（An ordered list of values）—-> 数组

1.3 形式

  • 对象：一个*无序*的键值对集合。一个以{开始，}结束。每个键值后跟一个『：』;键值对之间使用『,』分隔。

  • 数组：是值得*有序*集合，一个键可以对应一个数组。一个数组以[开始，以]结束。值之间使用『，』分隔。

  • 值：可以是双引号括起来的字符串、数值、true、false、null、对象或者数组。这些结构是可以嵌套的。

  • 字符串：是由双引号包围的任意数量Unicode字符的集合，适应反斜线转义。一个字符即一个单独的字符串。

  • 数值：数值与C/C++中的数值非常相似。

  • 空白：空白符号（包括空格、Tab以及回车）可以加入到任何符号之间，不会影响JSON的含义

二、JSON函数

  1. encode函数：将Python对象编码转换成JSON字符串
  2. decode函数：将已编码的JSON字符串解码成Python对象

四、Python –> JSON

4.1 基本语法参数

  1. JSON库中的dump以及dumps函数进行JSON编码

    • dump和dumps各项参数含义基本一致，唯一区别在于dump函数编码产生一个JSON文件流，而dumps编码产生一个JSON字符串

  2. dump 语法

    json.dump(obj, fp, skipkeys=False, ensure_ascii=True, check_circular=True, allow_nan=True, cls=None, indent=None, separators=None, default=None, sort_keys=False, **kw)

  3. dumps 语法

    json.dumps(obj, skipkeys=False, ensure_ascii=True, check_circular=True, allow_nan=True, cls=None, indent=None, separators=None, default=None, sort_keys=False, **kw)

  4. 参数详解

    • obj：编码对象
    • fp：指定文件流
    • skipkeys：若skipkeys为真，则非标准类型将会跳过，否则返回TypeError
    • ensure_ascii：若ensure_ascii为真，则输出时将会确保没有非ASCII字符；否则将会原样输出
    • check_circular：若check_circular为真，则将会检测是否存在循环调用；否则将会跳过检测
    • allow_nan：若allow_nan为真，则将会允许存在空值（无穷大值）；否则会返回ValueError异常
    • cls：若cls为控制，则会使用JSONError；否则使用指定子类进行编码
    • indent：若indent为空值，则无缩进；否则使用指定值进行缩进
    • separators：若separators为空值，则使用(‘,’,’:‘)进行分隔；否则使用指定的元祖分隔
    • default：若default为空值，则使用自带的default方法，否则使用指定值
    • sort_keys：若sort_keys为真，则输出将会按照键进行排序

4.2 参数实例

  • skipkeys参数

    • 如果key不是基本类型（*str、unicode、int、long、float、bool、None*），在dumps的时候会报TypeError错误。
    data = {1: 1, 'a': 'a', None: None, (0, 0): 'bad'}
# （0,0）：'bad'不是基本类型
json.dumps(data,skipkeys=True)
'{"a": "a", "1": 1, "null": null}'
json.dumps(data)
TypeError: keys must be a string

  • ensure_ascii参数

    • 所有的非ascii字符在输出时都会被转义为\uxxxx的序列，返回的对象是一个只由ascii字符组成的str类型。
    data = {u'游': u'戏', u'运': u'维'}
json.dumps(data)
'{"\\u6e38": "\\u620f", "\\u8fd0": "\\u7ef4"}'
json.dumps(data,ensure_ascii=False)
u'{"\u6e38": "\u620f", "\u8fd0": "\u7ef4"}'
print json.dumps(data)
{"\u6e38": "\u620f", "\u8fd0": "\u7ef4"}
print json.dumps(data,ensure_ascii=False)
{"游": "戏", "运": "维"}

  • indent参数

    • 设置对json进行pretty-printed的缩进空格数
    data = {'游': '戏', '运': '维'}
print json.dumps(data,ensure_ascii=False,indent=0)
{
"游": "戏",
"运": "维"
}
print json.dumps(data,ensure_ascii=False,indent=4)
{
    "游": "戏",
    "运": "维"
}

  • separators参数

    • 默认值是(‘,’,’:‘)，第一个元素为item的分隔符，第二个是key和value的分隔符。
    • separators=(',',':')是用来去掉json中的空格来压缩json
    print json.dumps(data,ensure_ascii=False)
{"游": "戏", "运": "维"}
# json压缩空格
print json.dumps(data,ensure_ascii=False,separators=(',',':'))
{"游":"戏","运":"维"}
# 替换分隔符
print json.dumps(data,ensure_ascii=False,separators=('-','+'))
{"游"+"戏"-"运"+"维"}

  • sort_key参数

    • 默认是false，不进行排序操作
    data = {'q': 1, 'w': 2, 'e': 3, 'r': 4}
# 默认不排序
print json.dumps(data, ensure_ascii=False, sort_keys=False)
{"q": 1, "r": 4, "e": 3, "w": 2}
# 排序
print json.dumps(data, ensure_ascii=False, sort_keys=True)
{"e": 3, "q": 1, "r": 4, "w": 2}

  • 命令行格式化显示json模块json.tool

    cat data.json
{"游": "戏", "运": "维"}
cat data.json | python -m json.tool
{
    "\u4e2d": "\u534e",
    "\u7231": "\u6211"
}

七、JSON –> Python

6.1 基本语法参数

  • JSON库中的load以及loads函数来进行JSON解码。
  • load和loads各项参数含义基本一致，唯一的区别在于load函数用于解码JSON文件流，而loads用于解码JSON字符串

6.2 load语法

json.load(fp, cls=None, object_hook=None, parse_float=None, parse_int=None, parse_constant=None, object_pairs_hook=None, **kw)

6.3 loads语法

json.loads(s, encoding=None, cls=None, object_hook=None, parse_float=None, parse_int=None, parse_constant=None, object_pairs_hook=None, **kw)

6.4 参数详解

  • fp：指定文件流
  • s；指定字符串
  • encoding：指定字符串编码
  • cls：若cls为空值，则使用JSONEncoder；否则使用指定子类进行编码
  • object_hook：若object_hook为空值，对象则返回解码器dict；否则使用自定义类型返回
  • parse_float：若parse_float为空，则JSON浮点数转换为浮点数；否则转换成指定类型
  • parse_int：若parse_int为空，则JSON整数转换为整数；否则转换为指定类型
  • parse_constant：若parse_constant为空，则不会处理非法输入；否则抛出异常
  • object_pairs_hook：若object_pairs_hook为空值，则数组返回解码器dict；否则使用自定义类型返回

6.5 参考实例

import json
json.loads('["foo", {"bar":["baz", null, 1.0, 2]}]')
['foo', {'bar': ['baz', None, 1.0, 2]}]
