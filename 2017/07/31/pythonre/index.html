Python Re模块

Jul 31, 2017 #Python

Re模块实现正则匹配操作，实现对复杂字符串截取有用信息。

一、Python RE 模块介绍

  • Python的 re 模块（Regular Expression正则表达式）提供各种正则表达式的匹配操作
  • 和Perl脚本的正则表达式功能类似，使用这一内嵌于Python的语言工具，尽管不能满足所有复杂的匹配情况，但足够在绝大多数情况下能够有效地实现对复杂字符串的分析并提取出相关信息。

二、Python 正则表达式语法

  • 正则表达式可以包含普通字符和特殊字符
  • 普通字符（比如数字或者字母）可以直接对目标字符串进行匹配
  • 利用特殊字符来模糊匹配某一些字符串的方法，比如『|』或者『(』使用这些特殊字符，正则表达式可以表示某一类的普通字符，或者是改变其周围的正则表达式的含义。

三、Python RE 主要功能函数

  • Python的 re 正则表达式模块定义了一系列函数，常量以及异常；同时，正则表达式被编译成『RegexObject』实例，本身可以为不同的操作提供方法。

3.1 compile函数

  • 语法
re.compile(pattern[,flags])
#  把正则表达式的模式和标识转化为正则表达式对象，供match()和search()这两个函数使用

  • re 所定义的 flag 包括:

    • re.l：忽略大小写
    • re.L：标识特殊字符集\w,\W,\b,\B,\s,\S依赖于当前环境
    • re.M：多行模式
    • re.S：即为『.』并且包括换行符在内的任意字符（『.』不包含换行符）
    • re.U：表示特殊字符集\w,\W,\b,\B,\d,\D,\s,\S依赖于Unicode字符属性数据库
    • re.X：为了增加可读性，忽略空格和’#‘后面的注释

  • 使用方法

    • 方法一
    import re
compiled_pattern = re.compile(pattern)
result = compiled_pattern.match(string)
    • 方法二
    compiled_pattern = re.compile(pattern)
result = re.match(compiled_pattern,string)
    • 方法一好处：能够实现正则表达式的重用

3.2 search 函数

  • 语法
re.search(pattern,string,flags=0)
# pattern：规则
# string：表示要匹配的字符串
# flags：标致位（开始匹配的位置），用于控制正则表达式的匹配方式
  • 使用方法
import re
compiled_pattern = re.compiled('a')
result = re.search(compiled_pattern,'abcdef')
print result.group(0)
'a'
# search 函数在字符串中查找匹配正则表达式模式的位置，返回 MatchObject 的实例，如果没有找到匹配的位置，则返回None

3.3 match 函数

  • 语法
re.match(pattern,string,flags=0)
# pattern：规则
# string：表示要匹配的字符串
# flags：标致位（开始匹配的位置），用于控制正则表达式的匹配方式
  • 使用方法
import re
name = "Hello,My name is huanghaihua,nice to meet you"
compiled_pattern = re.compiled('huanghaihua')
result = re.match(compiled_pattern,name)
print result.group(0)
# match 函数在字符串的开始位置尝试匹配正则表达式，也就是只报告从位置0开始的匹配情况

match函数与search函数差异

  • match 函数只在字符串的开始位置尝试匹配正则表达式
    • 如果字符串开始不符合正则表达式，则匹配失败，函数返回None；
  • search 函数是扫描整个字符串来超照匹配的
    • search 函数匹配整个字符串，直到找到一个匹配

3.4 split 函数

  • 作用：此功能很常用，可以将字符串匹配正则表达式的部分分割开并返回一个列表。

  • 语法

re.split(pattern,string,flags=0)
# pattern：规则
# string：字符串
# flags：最大分割字符串，默认未0，表示每个项都进行分割
  • 使用方法
words = "Words,words,words"
result = re.spilt('\W+',words)
print result 
['Words', 'words', 'words']
# \W：匹配任意非数字或者字母的字符

words = "Words,words,words"
result =  re.split('(\W+)',words)
print result
['Words', ',', 'words', ',', 'words']

words = "Words,words,words"
result = re.split('\W+',words,1)
print result
['Words', 'words,words']
# 1 ：表示最大分割次数

  • 特殊用法

    • 如果在字符串的开始或者结尾就匹配，返回的list将会以空串开始或者结尾
    words = "...words,words....."
result = re.split(''\W+,words)
print result 
['', 'words', 'words', '']
    • 如果字符串不能匹配，将会返回整个字符串的list
    words = "word,Word"
result = re.split('a',words)
print result 
['word,Word']

3.5 findall 函数

  • 作用：找到RE匹配的所有字符串，并把他们作为一个列表返回。这个匹配是从左往右有序地匹配，如果无匹配，则返回空列表。

  • 语法

re.findall(pattern,string,flags=0)
# pattern：规则
# string：目标字符串
# flags：规则选择项
  • 使用方法
import re
mail = '<user01@mail.com> <user02@mail.com> user04@mail.com'
compiled_pattern = re.compile('\w+@.....[a-z]{3}')
result = re.findall(compiled_pattern,mail)
print result
['user01@mail.com', 'user02@mail.com', 'user04@mail.com']
  • 如果无匹配，则返回空列表
import re
words = "bbbccc"
compiled_pattern = re.compile('a')
result = re.findall(compiled_pattern,words)
print result
[]

3.6 finditer 函数

  • 作用：找到RE匹配的所有子串，并把它们作为一个迭代器返回。这个匹配是从左到右有序地返回。如果无匹配，返回空列表。

  • 语法

re.finditer(pattern,string,flags=0)
# pattern：规则
# string：字符串
# flags：规则选择项
  • 使用方法
import re
compiled_pattern = re.compile('\d+')
it = re.finditer(compiled_pattern,"12a32bc43jf3")
for match in it:
  print match.group()
  
12
32
43
3

findall函数与finditer函数差异

  • findall 在字符串中找到正则表达式所匹配的所有字符串，组成一个列表返回。
  • finditer 在字符串中找到正则表达式所匹配的左右子串，并组成一个迭代器供遍历。

3.7 sub 函数

  • 作用：找到RE匹配的所有字符串，并将其用一个不同的字符串替换。

  • 语法

re.sub(pattern,repl.string,count=0,flags=0)
# pattern：规则
# repl：替换字符串，可以是字符串也可以是一个函数
# string：源字符串
# count：是模式匹配后替换的最大次数，count必须是非负整数
  • 使用方法
import re
repl = 'num'
words = 'one word two word three word'
compiled_pattern = re.compile('(one|tow|three)')
change = re.sub(compiled_pattern,repl,words)
print change
num word tow word num word

repl = 'num'
words = 'one word two word three word'
compiled_pattern = re.compile('(one|tow|three)')
changes = re.sub(compiled_pattern,repl,words,count=1)
print changes
num word tow word three word
# 指定count为1，只替换第一个

3.8 subn 函数

  • 作用：该函数的功能和sub()相同，但它还返回新的字符串以及替换的次数。

  • 使用方法

import re
repl = 'num'
words = 'one word two word three word'
compiled_pattern = re.compile('(one|tow|three)')
change = re.sub(compiled_pattern,repl,words)
print change
('num word tow word num word', 2)

四、补充知识点

4.1 在用正则表达式时尽量用原生字符串

在re模块里面，匹配一个反斜杠\需要两个反斜杠

  • 麻烦版
re.search('\\\\','fdsa\fdfs') 
在python中表示两个斜杠，其实就是一个斜杠
  • 精简版
re.search(r'\\','fdsa\fdfs')

4.2 正则表达式分组

  • 分组

    • group()：匹配到什么就是什么，所有的
    • groups()：匹配模式中出现分组的话，将分组的结果提取出来，返回给groups()
    • groupdict()：匹配模式中出现分组，将分组的结果提取出来，并与关键字组成字典，返回给groupdict()

  • 使用方法

import re
origin = "has fdsafdcdd"
r = re.match(h'\w+',origin)
print r.group()
has
print r.groups()
()

r = re.match('h(\w+)',origin)
print r.groups()
('as',)

r = re.match('h(?P<key>\w+)',origin)
print r.groupdict()
{'key': 'as'}

  • findall 分组

    • 使用方法
    origin = "fdhas fds hal err"
r = re.findall('h(\w+)',origin) 
# 直接将括号中的分组结果返回，只显示括号里的
print r 
['as', 'al']

origin = "fdhasaabc fds halaabc err"
r = re.findall('h(\w+)a(ab)c', origin)
# 先匹配，再将匹配的结果再进行匹配
print r
[('as', 'ab'), ('al', 'ab')]

  • spilt 分组

    • 使用方法
    origin = "hell0 test name com halt"
r = re.split('test', origin, 1)
print r   
# 有匹配但是没有取test
['hell0 ', ' name com halt']

r = re.split('(test)', origin, 1)
print r
['hell0 ', 'test', ' name com halt']
r = re.split('t(est)', origin, 1)
print r
['hell0 ', 'est', ' name com halt']
r = re.split('(t(es))t', origin, 1)
print r
['hell0 ', 'tes', 'es', ' name com halt']
