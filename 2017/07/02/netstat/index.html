Netstat Command

Jul 2, 2017 #linux

一个监控TCP/IP网络的非常有用的工具

1. 查看TCP连接状态

# 方法一
netstat -nat | awk '{print $6}' | sort | uniq -c | sort -rn

# 方法二
netstat -nat | awk '{print $NF}' | grep -v '[a-z]' | sort | uniq -c

2. 查找请求数前20个IP

# 方法一
netstat -anlp | grep 80 | grep tcp | awk '{print $5}' | awk -F":" '{print $1}' | sort | uniq -c | sort -nr | head -n20

# 方法二
netstat -anlp | awk '/:80/{split($5,ip,":");++A[ip[1]]}END{for(i in A) print A[i],i}' | sort -rn | head -n20

3. 用tcpdump嗅探80端口的访问看看谁最高

tcpdump -i eth0 -tnn dst port 80 -c 1000 | awk -F"." '{print $1"".$2"."$3"."$4}' | sort | uniq -c | sort -rn | head -20

4. 查找较多time_wait的连接

netstat -n | grep TIME_WAIT | awk '{print $5}' | sort | uniq -c | sort -rn | head -n20

5. 查找较多SYN的连接

netstat -an | grep SYN | awk '{print $5}' | awk -F":" '{print $1}' | sort | uniq -c | sort -nr | more

6. 根据端口列出进程

netstat -ntlp | grep 80 | akw '{print $7}' | cut -d/ -f1

7. 查看有多少个PHP-CGI进程活动

netstat -anp | grep php-cgi | grep ^tcp | wc -l

8. 查看PHP-CGI占用内存的总数

total=0
for i in `ps -C php-cgi -o rss=`
do
  total=$((${total}+${i}))
done
echo "PHP-CGI Memory usage:${total} kb"
